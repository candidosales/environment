#### Node

**FNM - Node.js version manager, built-in Rust**

https://github.com/Schniz/fnm

```shell

```

**NVM**

```shell
brew install nvm
```

**Node 20**

```shell

```

#### Ruby

**Rbenv**

```shell

```

**Ruby 3.3.5**

```shell

```
